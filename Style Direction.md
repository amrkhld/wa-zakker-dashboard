**Design Direction**

**Style**

* Light Theme
* Flat design
* Clean and minimal layout
* No gradients
* No textures
* No box-shadows
* No visual noise

---

**Colors**

Primary

* `#11538C`
* Used for buttons, links, clickable elements, icons, active states, indicators, and highlighted text

Secondary

* `#398CBF`
* Used for alternative buttons, hover backgrounds, selected items, tags, and soft highlights

Soft Highlight

* `#9AC1D9`
* Used for soft fills, subtle emphasis, and backgrounds behind icons or pills

Lines / Light Backgrounds / Cards Backgrounds

* `#F2F2F2`
* Used for subtle backgrounds, dividers, and card surfaces

Danger

* `#ed5e5e`
* Used for delete actions, errors, and warnings

Background

* White or transparent
* Keep most surfaces white

---

**Visual Rules**

* No shadows
* No glow effects
* No gradients
* No heavy outlines
* No borders
* Keep spacing generous
* Keep layouts simple

---

**Borders**

* No borders
* Border radius: `22px` (for pill-shaped and rounded-[22px] elements only)

---

**Typography**

Font

* Somar Sans

Language Style

* Arabic text
* English numbers

Text Style

* Clean and readable
* Avoid decorative styling

---

**Layout**

* RTL layout
* Elements aligned for right-to-left reading
* Consistent spacing
* Clear visual hierarchy

---

**Icons**

* Bootstrap Icons only
* Use simple line icons
* Use primary color for active icons
* Use default text color for inactive icons

---

**Components**

* Use Headless UI components
* Keep components simple
* Avoid decorative elements

---

**Interactions**

* Minimal hover effects
* Minimal active states
* Color change only
* No scaling
* No zooming
* No animation unless necessary

---

**Inputs**

* rounded-[22px] corners (22px)
* No borders (use background color or underline instead)
* Simple focus state using primary color
* No glow effects

---

**Buttons**

Primary Button

* Background: `#11538C`
* Text: White

Secondary Button

* Background: `#9AC1D9`
* Text: Primary color

Danger Button

* Background: `#ed5e5e`
* Text: White

---

**General Rules**

* Keep the interface calm and uncluttered
* Use color only when necessary
* Prefer whitespace over separators
* Keep UI predictable
* Avoid visual distractions
* Mobile  First
